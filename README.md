## To Run this Project:

### Requirements:
- [NodeJS](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/) or [NPM](https://www.npmjs.com/)

1. Clone this repo.
2. Go to project folder.
3. Run ```yarn``` or ```npm install```.
4. Run ```yarn start```
